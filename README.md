## GPT Demo

- Minimal GPT app 
- creates an AWS amplify project
- Deploys API Gateway
- Deploys Lambda function


### Functionality

- Acts as a video maker/creator
- helps generate instagram reel ideas

### How to run

- `amplify init`
- `amplify push`

[Note: add your open Ai key at amplify/backend/function/chat/src/handlers/openAi.js ]
